Are You Meshing With Us?

Florida Mesh is a community-driven, off-grid radio network built on Meshtastic. No cell towers. No internet required. Just LoRa radios linking neighbors across the state.

Get Started View the Mesh Map
What is this?

Long-range radio, no license required

Meshtastic turns inexpensive LoRa radios into a mesh network that relays messages node-to-node — even when the internet and cell service are down.

📡

Off-Grid Messaging

Send text messages, GPS coordinates, and telemetry over radio. Each node automatically relays for others, extending range across miles of Florida terrain.

⚡

No Infrastructure Needed

Works without cell towers, Wi-Fi, or internet. Solar-powered nodes can run indefinitely. When a hurricane takes down comms, mesh stays up.

🤝

Community First

An open, volunteer-run network. No subscriptions, no data collection. Neighbors helping neighbors build resilient communication across Florida.

Get on the mesh

Three steps to your first message

Getting started takes about 15 minutes and one affordable radio.

Get a Radio

Pick up a Meshtastic-compatible device — a Heltec V3, RAK WisBlock, or LilyGo T-Beam are popular choices. Most run $25–$60. Check the official hardware list.

Flash & Configure

Install the Meshtastic firmware, connect via the phone app or web client, and apply the Florida Mesh recommended settings so your node plays well with the network.

Get On the Air

Enable "OK to MQTT" in your LoRa settings to appear on the Florida Mesh Map and contribute telemetry. Then say hello on LongFast — you're part of the mesh.
